# Routes 目錄指南

## 概述

此目錄用於存放 API 路由定義。每個檔案通常定義一組相關資源的路由。

## 內容

- `authRoutes.js`: 認證相關路由。
- `recipeRoutes.js`: 食譜相關路由。
- `techNoteRoutes.js`: 技術筆記相關路由。
- `categoryRoutes.js`: 分類相關路由。
- `productRoutes.js`: 商品相關路由。
- `cartRoutes.js`: 購物車相關路由。
- `orderRoutes.js`: 訂單相關路由。
- `index.js`: 匯總所有路由。

## 路由規劃

- 參考專案計畫中的後端 API 端點規劃。
- 應用中間件進行認證和授權。
